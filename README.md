# 🚗 Car Chatbot Assistant

An intelligent multilingual chatbot for car information queries, supporting English, Arabic, and Tunisian dialects.

## Features

- 💬 Multilingual support (English, Arabic, Tunisian)
- 🔍 OCSVM-based intent classification
- 🚗 Car database queries (prices, availability, specs)
- 📊 Model comparisons
- 🌐 Smart link routing for non-car queries

## Tech Stack

- **Backend**: Flask, Python
- **ML**: scikit-learn, OCSVM
- **NLP**: deep-translator, langdetect
- **Frontend**: HTML, CSS, JavaScript

## Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR-USERNAME/car-chatbot.git
cd car-chatbot
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python app_flask.py
```

4. Open your browser: `http://127.0.0.1:5000`

## Usage Examples

**English:**
- "How much is a Toyota Corolla?"
- "Is BMW X5 available?"

**Tunisian:**
